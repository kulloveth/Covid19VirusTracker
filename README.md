# Covid19VirusTracker
An android app to keep track of covid 19 case statistics, News and show preventive measures

## Library Used
* Retrofit
* Gson
* Picasso
* WorkManager
* Coroutine 
* Koin
* ViewModel
* LiveData
* Navigation Component
* Room
* ViewBinding
